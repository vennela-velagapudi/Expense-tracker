# 💰 Expense Tracker

A command-line Personal Finance Tracker built using Python that helps users record income and expenses, manage monthly budgets, visualize spending patterns, and generate financial reports.

## Features

- Add income and expense transactions
- Store financial records using SQLite
- Set a monthly budget
- View monthly income, expenses, and savings
- Get alerts when the monthly budget is exceeded
- Visualize expenses using pie and bar charts
- Generate monthly and annual financial reports

## Technologies Used

- Python
- SQLite3
- Pandas
- Matplotlib
- Datetime

## How It Works

1. Run the Python program.
2. Choose an option from the menu.
3. Add income or expense records.
4. Set a monthly budget.
5. View summaries and reports.
6. Analyze expenses using charts.

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/personal-finance-tracker.git
```

Install the required libraries:

```bash
pip install pandas matplotlib
```

Run the application:

```bash
python finance_tracker.py
```
